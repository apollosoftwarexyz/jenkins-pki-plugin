Object Identifier (OID)

An Object Identifier (OID) (opens in a new tab) is a unique ASN.1 identifier for a field. All fields have an OID, but particularly well-known fields will be referred to by their alias (see above).

Common OID mappings:

Purpose Attribute Name Object Identifier (OID)
Email address E 1.2.840.113549.1.9.1
User ID UID 0.9.2342.19200300.100.1.44


For example:

The email address attribute OID is 1.2.840.113549.1.9.1 (typically aliased as E). So you could enter a mapping of 1.2.840.113549.1.9.1 for the OID, and E for the attribute name.
Then, if your certificates included an email address, you can enter E for the "Email Field" under TLS Certificate Attributes (see the section above) to use it.
