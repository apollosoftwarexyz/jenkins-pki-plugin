Attribute Name

This is an alias for the attribute that you can use when configuring attribute mappings.

  • Attributes are case-insensitive and will be normalized to uppercase.
  • If there is a well-known value (such as E for email address), you should preferably use it.
  • You should avoid using characters that require escaping, such as space, double quotes ("), ",", "+", "\", "<", ">" or ";".

Refer to the help information for Object Identifier (OID) above for further details.
