The filter used to locate users.

  • {0} is a placeholder for the Distinguished Name extracted from the user's X.509 certificate.
  • {1} is a placeholder for the Username extracted from the user's X.509 certificate (if there is one).
  • {2} is a placeholder for the Email extracted from the user's X.509 certificate (if there is one).

The default filter is DN={0}. A common alternative might be UID={1}
